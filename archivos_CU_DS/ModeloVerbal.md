# Modelo Verbal

Modelo Verbal

Se requiere desarrollar un sitio web que permita gestionar y consultar números de lotería junto con la fecha en que resultaron ganadores. El sistema deberá implementar las operaciones básicas de **CRUD (Crear, Leer, Actualizar y Eliminar)** para facilitar la administración de los registros.

La aplicación deberá contar con una interfaz gráfica atractiva, moderna e intuitiva, que permita a los usuarios navegar y realizar consultas de manera sencilla y eficiente. Además, se busca que el diseño sea amigable y adaptable para ofrecer una buena experiencia de usuario.

El desarrollo se realizará utilizando el framework **Django**, siguiendo la arquitectura **Modelo-Vista-Controlador (MVC)** para garantizar una adecuada organización del código, facilitar el mantenimiento del sistema y permitir su futura escalabilidad.
